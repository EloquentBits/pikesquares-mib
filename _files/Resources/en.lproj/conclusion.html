PikeSquares

Thank you for installing PikeSquares.

Run PikeSquares

Open a new terminal and run the following command to start PikeSquares:

  $ sudo launchctl start com.eloquentbits.pikesquares

Resources

Go through following links for additional information.

  • PikeSquares Cloud

Uninstall PikeSquares

Run the following command to uninstall PikeSquares.

  $ sudo /usr/local/bin/pikesquares-uninstall
